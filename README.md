# 2425os

## Description
A client-server system for indexing/managing text documents with metadata. Includes automated testing script.

## Prerequisites
- GNU Make
- GCC compiler
- Bash shell

## Installation
**Compiling**
```console
make clean && make
```
**Automated Testing files + Running Server**
```console
sh script.sh
```
**Client task**
```
./bin/dclient -<option> <args1,...>
```
